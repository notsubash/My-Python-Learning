# My Python Learning 🐍

A collection of Python exercises, notes, and code examples created while learning Python fundamentals.

This repository documents my journey through core Python concepts, covering data structures, built-in modules, functional programming basics, and exception handling.

## 📚 Topics Covered

| File | Topic |
|--------|--------|
| `01_lists.py` | Lists |
| `02_tuples.py` | Tuples |
| `03_dictionaries.py` | Dictionaries |
| `04_sets.py` | Sets |
| `05_strings.py` | Strings |
| `06_collections.py` | Collections Module |
| `07_itertools.py` | Itertools Module |
| `08_lambda_function.py` | Lambda Functions |
| `09_exceptions.py` | Exception Handling |

## 🎯 Purpose

The goal of this repository is to:

- Learn Python fundamentals
- Practice writing clean Python code
- Explore built-in data structures and standard library modules
- Build a strong foundation for software development, automation, data science, and machine learning

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/notsubash/My-Python-Learning.git
cd My-Python-Learning
```

### Run a file

```bash
python 01_lists.py
```

Replace the filename with any topic you want to explore.

## 🛠 Requirements

- Python 3.x

No external dependencies are required.
